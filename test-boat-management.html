🧪 Comment tester :

  1. Accédez à la page de gestion : boat-management.html
  2. Connectez-vous : Utilisez le bouton "Connexion" (les données de test fonctionnent)
  3. Testez la modification : Cliquez sur "Modifier" sur une carte de bateau
  4. Testez la suppression : Cliquez sur "Supprimer" (avec confirmation)
  5. Testez les images : Cliquez sur "Images" pour gérer les photos
  6. Testez l'ajout : Utilisez le bouton "Ajouter un Bateau"
